BMI Calculator – Python Project
Project Description :
This project is a Body Mass Index (BMI) Calculator developed using Python. It allows users to enter their weight (in kilograms) and height (in meters) and calculates their BMI. Based on the BMI value, the program classifies the user into health categories such as Underweight, Normal Weight, Overweight, or Obese.

Objective :
The main objective of this project is to understand user input handling, conditional statements, mathematical calculations, and error handling in Python.

Technologies Used :
Python
(Tkinter – if GUI version)

BMI Formula :
BMI = weight / height​

BMI Categories :
Underweight: BMI < 18.5,
Normal Weight: 18.5 – 24.9,
Overweight: 25 – 29.9,
Obese: BMI ≥ 30

Features :
User input validation
Accurate BMI calculation
Health category classification
Error handling for invalid inputs
(GUI interface – if advanced version)
